# 🛒 Nourshope Dashboard - Professional Task
**Frontend Development Task for Code Shefra (كود شيفرة) Internship**

### 📝 Project Overview
A modern, responsive e-commerce management dashboard built to demonstrate advanced frontend skills, including high-fidelity Figma-to-code conversion, state management, and API integration.

---

### 🚀 Technical Implementation
- **Figma to Code:** Precise conversion of design assets into functional React components.
- **Full CRUD Functionality:** Implemented Create, Read, Update, and Delete operations using RESTful API principles.
- **Advanced UI:** Integrated Dark Mode, fluid layouts, and interactive elements using Bootstrap 5 and FontAwesome.
- **Type Safety:** Developed using TypeScript to ensure robust and maintainable code.

---

### 🛠️ Tech Stack
- **Frontend:** React.js, TypeScript
- **Styling:** CSS3 (Custom Properties), Bootstrap 5
- **Icons & Animation:** FontAwesome, Animate.css
- **Backend Simulation:** JSON Server (REST API)
